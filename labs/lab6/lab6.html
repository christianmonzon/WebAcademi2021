For this lab, I had to learn how to use binary code and hexadecimal in order to create a favicon for my website. A favicon is an image that is used as a website icon. To create one, I used the website code.org where I would find the hex code of a color and add it there. I decided to create an electric guitar for my favicon because it is one of my favorite things to play. After learning how to use color pixalations, I then created my banner using the website Canva. To apply these images to my website, I first started to add my favicon. After I added my favicon after a few tries, I start adding my banner that I created beforehand. I set the banner to have a width of 100% and a height of 150px. From there, I add the older versions of my favicon as an image here on this lab, and here is where it ends.
