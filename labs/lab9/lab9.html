Explaining CSS in Depth
by Christian Monzon

What is Inline CSS?

Inline CSS gives us the ability to apply style rules to the HTML documents. Whenever you want to see a change or style a couple of elements, inline CSS is the best way to do so. Inline CSS helps you to style your HTML with colors, fonts and many more.

What is an Internal CSS?

When using the Internal CSS, it has to be written in the "head" section of your HTML. Using "styles" will allow you to alter the apperance of your document. That's how the Internal CSS is used to style a HTML page.

What is an External CSS?

The External CSS is CSS file separate from the others which can be found by by creating a link in the head of the webpage. The External CSS connects different webpages using the link of the stylesheet to apply the same styles to other webpages.
